## Introduction to constraint optimization


$$
\begin{aligned}
&\text{optimize} && f(x) \\
&\text{subject to} && c_i(x) \leq (\geq,=)\, 0, \quad i = 1, \ldots, m \\
& && x_j \in \mathcal{X}_j, \quad j = 1, \ldots, n
\end{aligned}
$$
